# Agenten-Orchestrierungs-Framework (AOF)

AOF ist ein offenes Framework für den Entwurf, Betrieb und die Governance von KI-Agenten in Unternehmen.

Es beschreibt KI-Agenten nicht als magische Assistenten, sondern als kontrollpflichtige Systemakteure: mit Rolle, Kontext, Regeln, Rechten, Verantwortung, Qualitätssicherung und Lebenszyklus.

## Grundsatz

Agenten folgen Rollen.  
Rechte folgen Aufgaben.  
Kritische Aktionen brauchen Kontrolle.
